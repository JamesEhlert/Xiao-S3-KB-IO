# Xiao-S3-KB-IO
Starter project for using the Seeed Studio Xiao ESP32-S3 board as a bluetooth keyboard dongle. A USB keyboard is connected to the USB-C port and the keystrokes are output through bluetooth.

This code is part of a Youtube series: ESP32 Adventures: https://www.youtube.com/playlist?list=PLzpQHur72fRwIrKFuXicBvjPrIdpV9_-o
